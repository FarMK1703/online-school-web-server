Доступные курсы

General English

  • Курс включает в себя следующие уровни:
  • Beginner ( Начинающий )
  • Elementary ( Элементарный )
  • Pre-intermediate ( Ниже среднего )
  • Intermediate ( Средний )
  • Upper-Intermediate ( Выше среднего )
  • Advanced ( Продвинутый )

Practice Speaking

  • Курс включает в себя следующие направления:
  • IELTS Speaking ( Разговорная речь для IELTS/CEFR )
  • General Speaking ( Общая разговорная речь )
  • Business Speaking ( Деловая разговорная речь )

Pre-IELTS English

  • Вводный курс IELTS включает в себя:
  • Ознакомление с форматом экзамена
  • Теоретические часть
  • Стратегии, советы и приёмы
  • Лексика

IELTS exam

  • Основной курс IELTS включает в себя:
  • Практическая отработка всех модулей на основе Cambridge IELTS Academic
  • Анализ ошибок и поэтапное их устранение
